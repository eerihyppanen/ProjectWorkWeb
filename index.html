Manual

Welcome to the data visualization page. Here's how you can use the page:

  • Map: Hover over the municipalities to see their names. Click on a municipality to update the chart with its data when municapality mode is selected.
  • Charts: Use the buttons below the chart to switch between line, bar, and pie charts(not all options are avaiable to every data type).
  • Data Selection: Use the radio buttons to switch between population data(just shows whole Finland's population) and municipality data.
  • Use order: select data mode and then chart type, note that changing chart type on municapilty mode resets data, you need to again select municipality
  • Exporting: By pressing the export button you can export the selected chart in SVG.

Enjoy exploring the data!
